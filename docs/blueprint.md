# **App Name**: LexiLeap

## Core Features:

- Progressive Mastery Training: An interactive learning circuit with three difficulty tiers (Easy Multiple Choice, Medium Fill-in-the-Blanks, and Hard Active Typing) based on Hebrew-English vocabulary weeks.
- The Soap Bubble Arcade: A gravity-based challenge where words fall in bubbles; players type Hebrew translations to 'pop' them, earning combo multipliers and regenerating hearts through accuracy.
- Fuzzy-Logic Spelling Assistant: An integrated Levenshtein distance tool that recognizes typos; it allows minor spelling errors on longer words and provides instant corrective feedback without halting progress.
- Hotkeyed UI Navigation: Full keyboard-optimized interaction using '1-4' for choices, 'Enter' for submission/continuation, and 'Escape' for navigation to ensure high-speed desktop gameplay.
- AI Practice Generator Tool: A generative tool that uses AI to analyze a user's Mistakes Bank and craft personalized practice sentences that incorporate multiple weak words into new, coherent contexts.
- Dynamic Mistakes Bank: A progress-tracking queue that uses localStorage to prioritize missed words; it requires a 2-point mastery increase before successfully 'graduating' a word from the bank.
- 3D Flip Flashcard Study: A study room featuring searchable vocabulary sets with interactive 3D-rotating flashcards, displaying English on front and Hebrew + glossaries on the back.

## Style Guidelines:

- The primary color is a sophisticated Royal Blue (#5865F2) to represent clarity and focus. The accent is a Bright Sky Blue (#5AB2FF), used for icons and highlights to provide an analogous and harmonious palette.
- The background uses an off-white Whispering Grey (#F4F7FB) with a hint of the primary hue for a high-end, contemporary learning feel.
- Feedback elements use Soft Peach (#FFA882) for errors and Mint Frost (#90E4C1) for success to maintain a friendly, gamified atmosphere.
- Headline font is 'Poppins', a geometric sans-serif that creates a fashionable and friendly appearance; 'PT Sans' is used for body text to add humanist warmth.
- Design features ultra-rounded corners (up to 24px) and 'chunky' button styles with soft 3D shadows to encourage interaction, consistent with top-tier education apps.
- Includes bouncy physics-based transitions for buttons and smooth sliding panel animations for feedback modals to emphasize a 'Living App' experience.
- Bold, soft-stroke iconography with rounded terminals that align with the geometric style of the 'Poppins' font.
